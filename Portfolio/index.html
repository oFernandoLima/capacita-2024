Interesses

Desenvolvimento Web

Desenvolvimento web é a prática de criar aplicativos e sites na web. Envolve a criação de interfaces de usuário, a implementação de funcionalidades e a integração com serviços e APIs.

Engenharia de Software

Engenharia de software é a aplicação de princípios de engenharia para o desenvolvimento de software. Envolve a análise de problemas, o design de soluções e a implementação de sistemas de software.

Microsserviços

Os microsserviços são uma abordagem de arquitetura de software que consiste em dividir uma aplicação em um conjunto de serviços pequenos e independentes, que se comunicam entre si para realizar uma tarefa.
